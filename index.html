Breve Gramatika

Nouns : Oftem -a bat bu sempre. Ex. : jiva, dwar, fish.

Adjectives : Oftem -e / -ney. Ex. : forte, sekret-ney

Adverbies : Oftem -em / -nem / -shem. Ex. : klarem, amiga-nem, respekti-shem.

Verbs : Oftem -i bat bu sempre. Ex. : fini, jan, yao.

Passate -te. Ex. : yu fini-te, ta jan-te, nu yao-te.

Passate Composit : he [verb]. Ex. : Yu he fini, nu he jan.

Future : ve [verb]. Ex. : Yu ve fini, ta ve jan, etc.

Conditionale : wud [verb]. Ex. : Yu wud fini, me wud jan.

Imperative : positive [verb] ba!, negative bye [verb]!

Participie Present : -she. Participie Passat : -ney. Gerund / Verbal Adverb : -yen.

Maskul : man- / -o. Femina : gin- / -ina. Pluri-forma : ~(e)s.

Nul defini-ney artikla. Adjectives invariabil.

Pronomines personal : Me, yu, ta (lu, ela), it ; Pluri-forma : Nu, yu, li.
